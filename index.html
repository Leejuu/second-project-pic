프리저브드 장미꽃을 제작할때 넣어 데코용으로 사용 할 수 있다.
입욕제를 뭉칠때 가장 많이 쓰는 반구틀과 꽃 틀 두가지 옵션을 두어
제작 시 선택의 폭을 넓힐 수 있도록 하였다.

 

입욕가루, 베이킹소다, 옥수수전분, 구연산

에센셜오일, 글리세린, 코코베타인

 

향과 거품을 내주는 제품들이다.

 

베이스에 뭉침을 주기위해 이 세가지를 적절히 사용한다.
원하는 향을 선택하여 즐거운 목욕시간이 될 수 있을 것 같다.
코코베타인은 배송이 잘 못와서 100ml공병에 따로 담아야 할 것 같다.

장점

 

1. 직접 손으로 만드는 과정이 재미있다.
2. 제작과정이 비교적 단순하고 쉽다.
3. 세가지 색상을 단독사용 또는 혼합하여 여러 색을 낼 수 있다.
4. 입욕제에서 원하는 향이 나서 좋다.
5. 천연재료이므로 피부에 무해하다.

 

 

 

단점

1. 생각보다 건조 시간이 오래걸린다.(최소5시간)
2. 조심한다해도 재료 특성상 다시 잘 깨진다.
3. 가루 날림이 있다.
4. 천연재료라 시중 입욕제만큼 풍성한 거품을 기대하기 힘들다.
5. 손재주가 많이 없다면 자꾸 부셔서 화가날 수도 있겠다.(내가 그랬다)

 

 

 

그럼에도

 

만드는 재미가 있고 원하는 모양과 향 그리고 색으로 만들 수있다는 장점이 있기 때문에 가치가 있다고 생각한다.
가격적인 면도 기존 브랜드, 키트 대비 경쟁력을 가질 수 있다고 판단된다.
